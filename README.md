# AI Ethics & Security Experiments

This repository is a collection of experiments where machine learning models are analyzed from an **AI Ethics and Security perspective**.

Instead of focusing only on model accuracy or performance, these experiments explore:

* Fairness and bias in ML models
* Feature influence on decisions
* Potential vulnerabilities in model behavior
* How models react to manipulated inputs
and many more..

Each experiment involves building a model and then critically evaluating it for **ethical and security risks**.

## Motivation

In my first semester of my **MS in AI**, I am studying two subjects:
**Machine Learning** and **AI Ethics**.

The AI Ethics course broadened my perspective, making me realise just how unfair and biased carelessly designed AI systems can get. At the same time, my interest in security and risk assessment makes me think about how AI models might be manipulated or attacked in real-world scenarios.
This led me to an idea: Why not practically implement the machine learning models taught in my ML course and evaluate them through both **ethical and security lenses**?

That idea became the motivation for this repository!!

This **AI Security Experiments Series** is my attempt to:

* Learn ML models
* Understand their decision-making behavior
* Identify possible ethical concerns
* Explore potential security risks


Each experiment typically includes:

* Model training
* Feature importance analysis
* Fairness evaluation
* Security / robustness testing
* Observations and insights

## Current Experiments

Decision Tree Model Audit

More experiments will be added as I progress through different ML models.


## Disclaimer

This project series is created **only for educational and ethical learning purposes**.

The goal is not to build perfectly optimized or production-ready ML systems but to expand my understanding of:

* Responsible AI
* AI security risks
* Model vulnerabilities
* Ethical implications of machine learning systems

All experiments are conducted on public datasets and are intended for learning and research.
